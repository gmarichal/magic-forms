# Formas Mágicas ✨

Juego web para niños: al hacer click o pulsar cualquier tecla aparecen formas de colores en pantalla que se desvanecen solas a los pocos segundos.

## Cómo jugar

- **Click**: aparece una forma en la posición del cursor. Si mantenés el click presionado, la forma va creciendo hasta 2.5x su tamaño original.
- **Tecla**: aparece una forma en una posición aleatoria de la pantalla.
- Cada forma suena con un efecto distinto al aparecer.
- El botón flotante 🔊 (arriba a la derecha) activa/desactiva el sonido.
- El fondo cambia de color cada 8 interacciones (colores claros, oscuros y degradados).

## Formas incluidas

Cuadrados, círculos, corazones, estrellas, triángulos, arcoíris, confites, flores y mariposas.

## Tecnología

HTML, CSS y JavaScript puro (sin dependencias ni build). Los sonidos se generan con la Web Audio API, sin archivos de audio externos.

## Correr localmente

Abrí `index.html` en un navegador, o serví la carpeta con cualquier servidor estático (por ejemplo `npx serve`).

## Deploy

Pensado para GitHub Pages: activá Pages sobre la rama principal apuntando a la raíz del repo.
